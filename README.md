# Power-BI-Dashboard-You-Tube-Performance-and-Audience-Engagement-Dashboard
Power BI Capstone Project
YouTube Performance & Audience Engagement Dashboard

This capstone project presents an interactive YouTube Performance & Audience Engagement Dashboard developed using Microsoft Power BI. The objective of this dashboard is to analyze channel performance metrics, audience engagement behavior, and category-level trends to generate actionable insights for content strategy and digital growth optimization.

Project Objective

The primary goal of this project is to:

Evaluate YouTube channel performance across multiple categories

Analyze subscriber growth and engagement metrics

Identify high-performing content types

Compare upload frequency versus audience reach

Support data-driven decision-making for content creators and media organizations

Business Insights

High upload frequency does not always guarantee proportional engagement.

Entertainment and Music channels show stronger subscriber retention.

Educational content has moderate engagement but lower viral potential.

News channels show high upload volumes but comparatively lower subscriber view concentration.

Audience engagement varies significantly by content category.

Technical Implementation

The dashboard was built using:

Power Query for data cleaning and transformation

DAX for calculated measures and KPIs

Interactive slicers for category-level filtering

Rank analysis visuals

Bar and comparative charts for performance benchmarking
